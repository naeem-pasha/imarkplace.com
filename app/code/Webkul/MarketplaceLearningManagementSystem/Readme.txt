#Installation

Webkul Marketplace Learning Management System module installation is very easy, please follow the steps for installation-

1. Unzip the respective extension zip and create Webkul(vendor) and MarketplaceLearningManagementSystem(module) name folder inside your magento/app/code/ directory and then move all module's files into magento root directory Magento2/app/code/Webkul/MarketplaceLearningManagementSystem/ folder.

Run Following Command via terminal
-----------------------------------
sudo apt install ffmpeg
composer require php-ffmpeg/php-ffmpeg
php bin/magento setup:upgrade
php bin/magento setup:di:compile
php bin/magento setup:static-content:deploy

2. Flush the cache and reindex all.

now module is properly installed
